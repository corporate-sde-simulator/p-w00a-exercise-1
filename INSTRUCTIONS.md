# Exercise 1: Python Fundamentals 🐍

> **Goal:** Learn the Python basics you'll need for ~30% of all tasks in this simulator.
> After this exercise, you'll be comfortable with variables, functions, classes, lists, dictionaries, and error handling.

---

## How This Exercise Works

1. Open `starter.py`
2. Read each section — it teaches you a concept with examples
3. Fill in the `TODO` sections (replace `pass` or `None` with your code)
4. Run the tests to check your work:
   ```bash
   python -m pytest test_exercise.py -v
   ```
5. Keep fixing until all tests pass ✅

---

## What You'll Learn

| # | Concept | Why You Need It |
|---|---------|-----------------|
| 1 | Variables & Types | Every program uses them |

## Tips

- **Read the examples first**, then try the TODO
- If you're stuck, the example right above the TODO shows you the pattern
- Don't worry about memorizing — you'll practice these in every task
- Run tests frequently: `python -m pytest test_exercise.py -v`
